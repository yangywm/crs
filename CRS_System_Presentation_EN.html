🚀 CRS - Claude Relay Service

  • 🔐Data Security: Self-hosted service ensures conversation privacy and security
  • 💰Cost Control: Share Claude subscription costs among team members

🔄 Data Flow

  • 👤User Requests → Cloudflare Pages (Frontend Interface)
  • 🔗API Calls → Cloudflare Workers (Business Logic)
  • 💾Data Storage → Cloudflare D1 (Persistent Data)
  • ⚡Cache Layer → Cloudflare KV (High-speed Cache)
  • 🤖AI Services → External APIs (Claude/Gemini/Codex)
